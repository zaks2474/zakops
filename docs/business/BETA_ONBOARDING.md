# Beta Onboarding Guide

Welcome to the ZakOps beta program! This guide will help you get started with the platform.

## Prerequisites

Before starting, ensure you have:
- [ ] Received beta access credentials
- [ ] Access to your organization's MSP data
- [ ] A modern web browser (Chrome, Firefox, Safari, Edge)

## Quick Start

### 1. Access the Dashboard

Navigate to your ZakOps dashboard URL (provided by your account manager):
- Production: `https://your-org.zakops.com`
- Demo: `http://localhost:13003` (local testing)

### 2. Initial Login

1. Click "Sign In"
2. Enter your beta credentials
3. Complete MFA setup if prompted
4. Accept the beta terms of service

### 3. First Steps

After logging in:

1. **Review Sample Deals**: Explore pre-loaded sample deals to understand the interface
2. **Create Your First Deal**: Use the "New Deal" button to create a test deal
3. **Try Agent Suggestions**: Enable AI assistance to see deal recommendations
4. **Set Up Approvals**: Configure approval workflows for your team

## Feature Overview

### Deal Management
- Create, edit, and track deals through their lifecycle
- Attach documents and notes
- Set reminders and deadlines

### AI Agent Integration
- Automated deal analysis
- Risk assessment suggestions
- Pricing recommendations
- Contract clause analysis

### Approval Workflows
- Multi-level approval chains
- HITL (Human-in-the-Loop) verification
- Audit trail for all decisions

### Reporting
- Deal pipeline dashboard
- Win/loss analysis
- Performance metrics

During beta:
- Maximum 100 deals per organization
- Agent analysis limited to 50 requests/day
- Some features may change based on feedback
